# AI Based Job Recommendation Portal

A frontend-based job recommendation portal developed using HTML, CSS, and JavaScript.  
The project provides a user-friendly interface for exploring job opportunities, managing dashboards, and interacting with company-related sections.

## Features
- Responsive user interface
- Job recommendation section
- Company dashboard pages
- Interactive frontend design
- Clean and modern layout

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Status
Currently working on enhancing the project using React.js and API integration.
